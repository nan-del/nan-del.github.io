IP地址的溯源

为何要溯源IP地址

1、在互联网上的违法犯罪

很多人以为在互联网上可以为所欲为，想干什么就干什么，因为以为没人可以找他，于是可以在网上随意发表言论、散播谣言、传播不正确的思想，更有甚者，从事违法犯罪行为，如收集、贩卖用户的个人隐私信息，诈骗等等。但其实我们可以利用IP地址识别那些在网上从事非法活动的人，以便追究他们的责任。

当然有人会说，你怎么追踪IP地址呢？当只有一个公网地址的时候，你可以很轻松的定位到IP地址，但是一旦进入内网，就像鱼入大海一样，难道你还能从成千上万的鱼中找出一条鱼吗？确实，一般的个人或者大一点公司要从成千上万的网民中，找出一个犯罪的网民是很难的，但不代表其他人做不到。所有的上网行为都绕不过ISP(即互联网服务提供商)，所以，你的所有上网行为其实都会被ISP记录，但是ISP不会轻易发布这些信息，大多数互联网服务提供商都遵守严格的隐私规定，所以普通人索取信息不太可能成功。但是如果执法代表或者版权律师有证据的话，他们就会被区别对待。

如果一个人在网上销售或传播非法的东西，或者在网络平台上谈论犯罪，执法部门可以要求你的 ISP 提供他们的个人信息而且网络服务提供商很可能会合作并提供这些信息。这样，你觉得你还能跑的掉吗？

同时IP 地址也可以用于识别垃圾邮件和网络钓鱼邮件背后的罪犯。电子邮件客户端和电子邮件营销软件平台根据电子邮件的内容和主题行，以及收件人点击“标记为垃圾邮件”按钮的行为，记录哪些电子邮件地址看起来像垃圾邮件。虽然他们没有能力找到电子邮件地址背后的个人，但是他们可以把标记的 IP 地址添加到一个黑名单，以达到从该地址发出的电子邮件都会被标记为垃圾邮件。

2、市场影响

在涉及到违法犯罪的事件，IP地址可以追踪到个人，但是当涉及到市场使用时，IP地址更多的是提供一个大致的信息，并不会知道你的名字、家庭住址等隐私信息。

当你的 IP 地址连接到一个服务器，以访问一个特定的网站，网站可以跟踪访问者来自哪里。例如，当你在淘宝搜索一些关键词的时候，页面通常会自动检测你来自哪里。这些都是基于IP地址得到你的大致位置。如果网站使用 了cookies的话，这些信息包可以跟踪和收集关于你网站行为的额外信息，它们也可以将你的访问与关于你的其他数据联系起来。例如，可以看到你是否是第一次访问这个网站，或者你是否返回，以及你访问了哪些特定的页面。如果你看到你以前浏览过的网站的广告跟踪你的网络活动，那是 cookies 跟踪你网络活动的结果。虽然你的 IP 地址提供了你的位置信息，但是 cookies 为网站和广告商提供了更多关于你在线行为的详细信息。

3、诈骗侦测

现在支付宝和一些银行都会使用这样一种技术来侦察盗用他人银行卡的行为，如果使用信用卡购买的地点与信用卡持有人居住的地点不同，或者距离上一次使用信用卡的距离太长，就比如上一秒你可能在广州买了一瓶水，但下一秒你就在北京买了一个10000块的包，这样银行或者支付宝公司就会在处理信用卡之前与持卡人核对信息，是否是本人消费的。这种机制可以有效的防止一部分的欺诈行为。

怎么防止IP的追踪

1、使用VPN专用网络

使用虚拟专用网络(VPN)，当你浏览网页时，它会屏蔽你的 IP 地址。它加密你所有的互联网活动和屏蔽网站识别你的地理位置。对于任何关心网络隐私的人，或者那些希望绕过访问网站的地理限制的人来说，VPN 服务都很方便。

2、使用代理服务器

Vpn 不是免费的，所以如果你想要某种程度的 IP 网站跟踪保护，但又不想花钱，你也可以选择代理服务器。代理服务器通过使用中间人 IP 模糊您的 IP 地址。它不像 VPN 那么安全，因为它不为你的数据提供加密，但是它确实可以防止你的 IP 地址被普通网站用户访问。

3、使用浏览器插件

Tor 是一个免费的开源浏览器附加组件，每次你访问一个网站时，它都会将你的网络连接弹出几个不同的节点，使你的原始 IP 地址几乎无法被追踪。它不像免费服务所期望的 VPN 那样安全，但它提供了额外的加密和匿名性。（但我没用过，之后可以尝试一下）

总结

其实对于一般人来说，IP地址泄露是不需要太担心的事件，对于一些网站虽然知道你的位置信息，但是并不会泄露太多个人信息，反而对于一些时候，IP跟踪反而会为我们提供一些便利，但是对于一些遭遇黑客攻击的公司来说，IP跟踪可以提高黑客攻击的代价，可以让攻击行为不会太张狂。因为现在的法律法规对一些违反犯罪的检测力度和惩罚力度都在加大，这样一旦黑客被抓到，就将面临严厉的惩罚，这就会让一些黑客收敛一些。
